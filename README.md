## Shortcuts

### JetBrains IDE (CLion, IntelliJ IDEA, PyCharm, ...)

- Toggle line comment
  - macOS : `Command + /`
  - Windows : `Control + /`
- Format code
  - macOS : `Option + Command + L`
  - Windows : `Control + Alt + L`

### Visual Studio (not Visual Studio Code)

- Toggle line comment : `Control + K + /`
- Format code : `Control + K + D`
